# Aura Kinetic

Aura Kinetic is an interactive web experience built with modern web technologies, featuring 3D rendering and smooth animations.

## Features

- **3D Rendering:** Uses `@react-three/fiber` for declarative 3D scenes.
- **Smooth Animations:** Integrated with `@tweenjs/tween.js` for fluid, high-performance transitions.
- **Optimized:** Fast and responsive, making use of modern build tools and lightweight packages.
